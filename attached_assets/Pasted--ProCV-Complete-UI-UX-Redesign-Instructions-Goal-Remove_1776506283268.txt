## 11. DESIGN CONSISTENCY — EVERY PAGE MUST LOOK THE SAME

This is the most important rule of all. Right now different pages/components
look like they were built by different people on different days. That is
the second biggest AI tell after the purple color. Fix this.

### The Golden Rule
> Every single page, component, card, button, and input in the entire app
> must feel like it came from the same designer, on the same day,
> using the same rulebook. A user should NEVER feel like they changed apps
> when they click between tabs.

---

### 11.1 — Page Layout Must Be Identical Across All Views

Every view (CV Generator, Job Board, CV Toolkit, Scholarship, History, Tracker)
must follow this EXACT same layout structure:

```
┌─────────────────────────────────────────┐
│  Page Title (Playfair Display, 22px)    │
│  Subtitle (DM Sans, 13px, muted gray)   │
│  ── Gold accent line (40px wide) ──     │
│                                         │
│  [ Card ]                               │
│  [ Card ]                               │
│  [ Card ]                               │
└─────────────────────────────────────────┘
```

Apply this to EVERY view:
```tsx
// Use this header block on EVERY page — copy paste it
<div style={{marginBottom: '20px'}}>
  <h2 style={{fontFamily: "'Playfair Display', serif", fontSize: '22px',
              fontWeight: 700, color: '#1A1A1A'}}>
    Page Title Here
  </h2>
  <p style={{fontSize: '13px', color: '#6B7280', marginTop: '4px'}}>
    Short description of what this page does
  </p>
  <div style={{width: '40px', height: '2px', background: '#C9A84C',
               marginTop: '8px', borderRadius: '2px'}} />
</div>
```

---

### 11.2 — Card Rules (Apply Everywhere, No Exceptions)

Every card across every component must look like this:
```tsx
// THE ONLY card style allowed in this app
<div style={{
  background: '#FFFFFF',
  borderRadius: '12px',
  border: '0.5px solid #E5E2DC',
  padding: '20px',
  marginBottom: '16px'
}}>
```

❌ NEVER use different border radius on different pages (some 8px, some 16px, some xl)
❌ NEVER use different padding on different pages (some p-4, some p-6, some p-8)
❌ NEVER use dark card backgrounds on some pages and white on others
❌ NEVER use shadow on some cards and no shadow on others

---
